# Frontend Functional Test Checklist

이 문서는 Salon Manager 프론트엔드(Admin Web)의 주요 기능에 대한 테스트 시나리오와 검증 방법을 정의합니다.
현재 구현된 기능과 구현 예정인 기능(선불권 결제 등)을 모두 포함하여, 통합 테스트 시 활용하시기 바랍니다.

## ✅ 테스트 방법 (Testing Methods)

1.  **매뉴얼 테스트 (Manual Testing):**
    *   Chrome 개발자 도구(F12)를 열어놓고 진행합니다.
    *   `Network` 탭에서 API 요청/응답 실패(Red color)를 확인합니다.
    *   `Console` 탭에서 자바스크립트 에러 발생 여부를 확인합니다.

2.  **데이터 일관성 검증:**
    *   프론트엔드에서 변경한 내용(예: 예약 생성)이 새로고침 후에도 유지되는지 확인합니다.
    *   필요 시 백엔드 DB(`mysql`)를 직접 조회하여 데이터가 정확히 저장되었는지 교차 검증합니다.

## 🚀 테스트 실행 가이드

1.  **터미널 1 (Backend):** `yarn dev:backend` (Port 3000)
    *   *Ready 상태 확인 (NestJS Application Successfully Started)*
2.  **터미널 2 (Frontend):** `yarn dev:frontend` (Port 5173)

4.  위 시나리오를 순서대로 수행하며 검증합니다.
